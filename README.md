Victor's Dotfiles
=================

This is a repository for bootstrapping my dotfiles with [Dotbot][dotbot].
It was created using the suggested [template][template]

In general, you should be using symbolic links for everything, and using git
submodules whenever possible.

To upgrade submodules to their latest versions, you can periodically run
`git submodule update --init --remote`.

[dotbot]: https://github.com/anishathalye/dotbot
[template]: https://github.com/anishathalye/dotfiles_template/generate
